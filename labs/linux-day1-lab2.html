sort — Ordering Output
Exercise 5 — Sort a List Alphabetically
Sort the lines of a file A to Z
SORT
Key Concept

sort sorts lines of a file alphabetically by default. sort -r reverses (Z→A). sort -n sorts numerically instead of alphabetically (important — without -n, "10" sorts before "9"). Almost always paired with uniq.

uniq — Deduplication
Exercise 7 — Remove Duplicate Entries
Chain sort and uniq to get unique lines only
UNIQ
Key Concept

uniq removes adjacent duplicate lines. This means it only works correctly after sorting — because duplicates must be next to each other to be detected. Always use sort | uniq together, never uniq alone.

tr — Transform Characters
Exercise 9 — Normalize Text to Uppercase
Convert all lowercase characters to uppercase
TR
Key Concept

tr translates or deletes characters. It reads from stdin (pipe), not files directly. tr 'a-z' 'A-Z' converts lowercase to uppercase. tr -d '\n' deletes newlines. Used often in shell scripts to normalize text before processing.

A shell script is comparing environment names but they're inconsistently cased — sometimes "production", sometimes "Production". You want to pipe the content of env_name.txt through tr to convert everything to uppercase for consistent comparison.
